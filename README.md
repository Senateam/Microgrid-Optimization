# Microgrid Optimization: A Synergy of PV Systems and Demand Side Management

This repository contains MATLAB scripts and sample data for the project **"Enhancing Microgrid Efficiency: A Synergy of PV Systems and Demand Side Management"**, presented at the IEEE International Conference on Electro Computing Technologies and published on IEEE Xplore.

## Overview
The project investigates how photovoltaic (PV) systems integrated with demand-side management (DSM) strategies can **optimize energy distribution, minimize losses, and enhance microgrid efficiency**. The simulation models a smart microgrid with solar generation, loads, and energy management algorithms.

## Repository Structure
- `scripts/` - MATLAB scripts and functions for simulation and optimization
- `data/` - Sample load profiles and PV generation data
- `results/` - Generated plots and output files
- `docs/` - Optional detailed report or research summary
- `README.md` - This file

## Requirements
- MATLAB R2022a or later
- Optimization Toolbox
